ABOUT CSS

CSS stands for cascading styles sheet, it is used to add styles to a webpage.
It can be either internal or external or inline

External CSS

This is a where a separate file is created with extension .css and the elements in the html document is target using either an id, class or even the elements name itself. The html file is then linked to this css file

Internal CSS

This is a where the style element is used in the html document and the different elements are targeted using their classes, id and even their element names

Inline CSS

This is a where the style attribute is used. This can only style a particular element
